HTML, CSS, and DOM analogy.

HMTL is like the foundations for a house. HTML gives you the structure for a webpage. CSS is then the styling of the house, for example what wallpaper or paint is on the walls. The DOM is a tool that lets us interact with all the elements, the structure and contents of a page.

What is meant by boxifying design?

Boxifying design is a helpful tool to use when faced with the task of turning a conceptual design for a website into the real thing. By dividing up the file into boxes one can start to determine and breakdown the layout of the HTML file and where different elements will go. It is helpful to start with the biggest of these elements and work down to the smallest.

What is the box model?

The box model describes how different boxes on a HTML page associate with each other and determines the basic layout of a page. The boxes associate with each other using four distinct parts, which are (starting from the inside and working out) the content, padding, border and margin.
